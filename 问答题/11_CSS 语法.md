## 1. 什么是 CSS？它有什么优势？
CSS是层叠样式表的英文简写，它用来定义网页的样式。

CSS的优势是：提供了灵活的对样式的控制，使网页的内容和表现分离，让代码更简洁，可维护性更好。

## 2. 内部样式表和外部样式表各是如何声明和使用？
内部样式表在`<head>`中声明，例如：
```html
<style type="text/css">
  body{
    color: yellow;
  }
</style>
```
使用方法是在body中使用样式表中定义的样式，如果css中直接对某一标签进行声明，则使用标签的同时就自动使用了这个样式。

外部样式表在一个外部的css文件中声明，声明和使用方法都相同，但需要在html文件中对css文件进行链接，例如：
```html
<link rel='stylesheet' type='text/css' href='main.css'>
```

## 3. 多层样式层叠是什么意思？
多层样式层叠是说，css可能在相同或不同的文件中、以相同或不同格式被多次定义。当一个html元素同时使用了多个样式表时，这些样式表会以一定的优先次序层叠为一个样式表，供这个html元素使用。

## 4. 什么是优先级？样式表的优先级排序是什么？
优先级就是指当多个样式表发生冲突的时候（也就是对同一个元素声明了同一个属性，但属性的值不同），决定了哪个样式表会成为这个元素的最终样式的参数。

优先级排序，从高到低是：内联样式 > 内部样式 > 外部样式 > 浏览器缺省设置。

## 5. 请解释 CSS 如下概念：
* 选择器：这个样式会作用于哪一类元素。
* 属性：要设置这个元素的哪一种样式。
* 值：这个属性将被设置的值。
* 声明：对一个属性的设置。
* 声明块：一些声明包含在同一个大括号内，形成声明块。

## 6. CSS 中回车符(\n)会影响 CSS 的解析吗？请测试在不同位置下的回车符对解析的影响，然后谈谈你的理解
如果回车不中断选择器、属性或者变量的名称，则没有影响。如果中断了，浏览器就无法解析这些名称，导致出错。

## 8. 浏览器前缀是什么？我们为什么需要浏览器前缀？
是跟在属性之前的前缀，加起来形成一个新的属性。格式形如`-浏览器内核简称-`，例如webkit内核下的transform属性就是`-webkit-transform`。

因为有的属性一些浏览器会使用，但由于不在html标准中，有的浏览器不支持。为了在支持这个属性的浏览器中使用这个属性，在不支持的浏览器中不使用，就要加上这个前缀，表示这是供某个特定的浏览器使用的属性。

## 9. 对浏览器前缀的处理有哪些方案？其中较好处理方案是哪个？
* 为每种浏览器内核各写一个前缀。
* 不写前缀，放弃对不支持这个属性的浏览器的兼容。
* 不写前缀，然后使用css预处理器自动加上。

比较好的处理方案是最后一个。

## 10. CSS 缩写是什么？哪些属性支持缩写？
CSS缩写就是指，一些表达可以用更短、更简便的方式替代，而含义不变。

* 颜色可以用三位代替六位，此时这六位两两相等，例如#223344缩写就是#234.
* property，border，background，font下属包含了好几个属性，而这些属性通常一起设置，例如边框的上下左右。为了简便，可以将这些属性用空格隔开，直接排列在总属性名的后面。例如`border: 1px solid #000;`就指定了3个属性，分别是边框粗细、样式和颜色。

## 11. 什么是 CSS 中的继承？哪些属性可以继承？如果不希望子元素继承父元素的某个属性该怎么办？
由于html是树形结构，父元素中会包含子元素，而对父元素指定的一些样式，也会在父元素下的子元素中同时使用，也就是子元素“继承”了父元素的样式。

和文本、列表相关的属性，以及颜色可以继承。

如果不希望子元素继承父元素的一些属性，就把同样的属性在子元素中重新声明一遍，写成子元素应有的值。

## 12. 以下代码&lt;h2>最终显示成什么颜色？为什么？

```html
<!DOCTYPE html>
<html>

<head>
<style type="text/css">
  body{
    color: yellow;
  }
</style>
</head>

<body>
<h2>标题</h2>
</body>

</html>
```

显示成黄色。因为在内部样式表中指定了body中所有元素都显示成黄色。

## 13. 以下代码中&lt;h2>最终显示成什么颜色？如果我想让color: red，在不修改内部和外部样式表的前提下，该如何实现？

index.html

``` html
<!DOCTYPE html>
<html>

<head>
 <link rel="stylesheet" type="text/css" href="style.css">
 <style type="text/css">
   body{
     color: yellow;
   }
   h2 {
     color: blue;
   }
 </style>
</head>

<body>
 <h2>标题</h2>
</body>

</html>
```

style.css
``` css

h2 {
 color: green
}

```

显示成蓝色。

实现方法：
将body中的`<h2>标题</h2>`改成`<h2 style="color:red">标题</h2>`。
